# 32 前端滚动与同步面板折叠迭代记录

日期：2026-06-11

## 需求来源

用户反馈：

1. 左侧邮件列表和右侧邮件详情/ Todo 页面绑定滚动，浏览体验不舒服。
2. 邮箱同步区域通常只在开始使用时需要，后续更多操作集中在点击邮件和查看 Agent 结果，因此希望同步区域可以折叠。

## 已完成修改

- 页面改为固定高度的工作台布局。
- `body` 不再作为主要滚动容器。
- 左侧栏 `mail-sidebar` 独立滚动。
- 右侧邮件详情页通过 `tab-body` 独立滚动。
- 右侧 Todo 总览通过 `todo-board` 独立滚动。
- 邮箱授权区域增加“收起/展开”按钮。
- 折叠状态会保存在浏览器 `localStorage`。
- 同步并自动运行 Agent 成功后，邮箱授权区域会自动收起，把空间让给邮件列表。

## 验收点

1. 滚动左侧邮件列表时，右侧邮件详情保持不动。
2. 滚动右侧邮件详情、日志或 Todo 总览时，左侧栏保持不动。
3. 点击“收起”后，邮箱授权区域只保留标题和展开按钮。
4. 点击“展开”后，可以继续修改邮箱同步配置。
5. 同步成功后，同步区域自动折叠。
