# 答辩演示脚本

1. 打开系统首页，介绍智能邮箱 Agent 的目标。
2. 点击“导入样例邮件”，展示邮件列表。
3. 选择任务邮件，点击“运行 Agent”，展示分类、Todo、草稿和 Critic 审查。
4. 选择附件邮件，运行 Agent，展示 CSV 附件分析和表格任务转 Todo。
5. 在 Todo 区域点击确认或完成，展示人机协同。
6. 打开 Agent 日志，说明 Triage、Action、Critic 的协作过程。
7. 展示 docs 目录，说明敏捷启动、计划、执行和验收过程文件。
8. 展示真实邮箱测试指南，说明系统不仅能跑样例，也能通过 IMAP 同步真实邮箱。
